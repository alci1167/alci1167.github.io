Lab 7 - “Trust the Process-ing”

DESCRIPTION

In this lab we will learn how to create interactive animations and visualizations on our computers using Processing, and then connect them to the Arduino to control them using electronics

MATERIALS

  • Arduino (Elegoo) Uno R3 Controller Board
  • USB Cable
  • Breadboard Jumper Wire
  • Joystick
  • Multimeter

PART 0: SETTING UP PROCESSING

We will begin by setting up Processing. We can go to the Downloads page of Processing at Processing.org and download the software for our operating system.

PART 1: CREATING WITH PROCESSING

When we open Processing on our computer, it should have opened a PDE window with an empty Text Editor. In there we can use some sample code, and press the Run button on the top left of the PDE to run the program.

We can change the color and size of the ellipse drawn and see how it changes the behavior of the program. I chose to make the circle smaller and filled in orange.

PART 2: PROCESSING WITH ARDUINO

In this part of the lab, we will first set up our Arduino to obtain data from the joystick, and then send it over to the Processing sketch such that we can control the circles using the joystick.

We will use the joystick module connected to the Arduino, along with its in-built push button which gets pressed when you press down on the joystick. After connecting the joystick to the Arduino, we can write a sketch to read the X and Y positions of the joystick, and also whether the push button on the joystick is pressed. Once we have read the three values of interest from the joystick (X position, Y position, and button), we can pack it together in a format that we can send over serial to be able to be read by Processing.

We can now move to writing the Processing sketch that will take these joystick values from the Arduino and update the animation. In order to test the code, we first upload the Arduino sketch to the Arduino board. Then, while the board is connected to the computer, run the Processing sketch using the run button.

PART 3: CREATING YOUR OWN VISUALIZATION

Now that we know how to use Processing with Arduino, we can design a new Arduino-controlled visualization ourselves. We can create a complete interactive visualization on Processing that connects with the Arduino and is controlled by 3 input electronic components. I used a button to change the size of a circle, a potentiometer to change the color of the circle, and a photoresistor to change the background color.

CONCLUSION

This lab was interesting and interactive. I really enjoyed having a visual component with the sensors. The processing side of things was a bit intimidating at first, but reading the lab definitely helped. The hardest part of this lab was working with the sensors. I was having trouble getting them to give accurate readings, which made working with the data a bit harder. Overall this lab was enjoyable and helped me get more comfortable with the electronics.
