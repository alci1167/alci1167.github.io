Lab 1: Simple Circuits

DESCRIPTION

This lab is an introduction to electronics and simple circuits.

MATERIALS

  • Power Supply Module
  • Button
  • 830 Tie-Points Breadboard
  • Breadboard Jumper Wire
  • 9V Battery with Snap-on Connector Clip
  • Resistors
  • LEDs
  • Multimeter

PART 1: SETTING UP POWER

Using the power supply module, 830 Tie-Points Breadboard, Breadboard Jumper Wire, and the 9V Battery with Snap-on Connector Clip we can use the multimeter to test whether we are getting the expected power output on either side. The images below show how this was set up for both the 3.3V and 5V ouputs.

PART 2: (NOT) BLOWING UP AN LED

3.3V Output

We are powering the LED with 3.3V source and our LED has a current of about 20 mA - 35 mA. Using Ohm's Law: R = 3.3V/30mA = 110 Ohm. This tells us we should use a 110 Ohm resistor. We have a 100 Ohm resitor that should work. The images below shows the schemtic and circuit.

5V Output

The LED has a current of about 20 mA - 35 mA. Using Ohm's Law: R = 5V/30mA = 166.66 Ohm. This tells us we should use a 166 Ohm resistor. We have a 100 Ohm resitor and a 220 ohm resistor. With 100 Ohm the current would be 50 mA and with the 220 Ohm resistor the current would be 22 mA. The image below shows the schematic and circuit with a 220 Ohm resistor.
