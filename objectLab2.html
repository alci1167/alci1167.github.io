Lab 2: Digital IO

DESCRIPTION

This lab is an introduction to the Arduino, its Integrated Development Environment (IDE), and digital I/O on the Arduino. We will set up the workflow of programming an Arduino and learn about how to manipulate and sense its digital pins.

MATERIALS

  • Arduino (Elegoo) Uno R3 Controller Board
  • USB Cable
  • Button
  • 830 Tie-Points Breadboard
  • Breadboard Jumper Wire
  • Resistors
  • LEDs
  • Multimeter

PART 1: SETTING UP THE ARDUINO IDE

We will be using the Arduino microcontroller board, and we will be programming it using the Arduino IDE. Using the IDE, we will write and upload code to the Arduino board. The first thing we are going to do is run the Blink example file.

PART 2: BLINKING AN EXTERNAL LED

Since We can control Arduino pins to output a certain voltage, we can use it to control an LED. We can take an LED and a resistor and connect them to the Arduino such that the LED is powered by the digital pin 10. We can do this by changing what pin we output to.

PART 3: BUTTON PRESS AND SERIAL MONITOR

Similar to how we set up a digital pin to be an OUTPUT in the previous part, we can also set one up to be an INPUT. This means that we will be able to detect whether the pin has HIGH or LOW voltage. We can push a button connected to a digital input with a resistor to ground and add this to circuit to the LED circuit.
